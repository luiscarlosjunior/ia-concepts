# Documentação de Estatística

Esta pasta contém a documentação teórica relacionada aos conceitos de estatística implementados nos notebooks.

## Conteúdo

### Teoria (`/teoria`)
Documentação detalhada sobre conceitos estatísticos e métodos numéricos:

- **`probabilidade_basica.md`**: Conceitos fundamentais de probabilidade
- **`medidas_tendencia_central.md`**: Média, mediana, moda e outras medidas
- **`medidas_dispersao.md`**: Variância, desvio padrão e dispersão
- **`ajuste_de_curva.md`**: Técnicas de ajuste e regressão
- **`metodos_numericos_mmq.md`**: Método dos Mínimos Quadrados
- **`metodos_numericos_sistemas_lineares.md`**: Solução de sistemas lineares
